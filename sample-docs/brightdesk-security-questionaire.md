# BrightDesk Support Security Questionnaire

## Vendor Overview
BrightDesk Support provides customer support ticketing and shared inbox software for customer success teams.

## Data Processed
BrightDesk may process customer names, email addresses, support messages, attachments, account IDs, and internal support notes.

## Authentication
BrightDesk supports single sign-on using SAML 2.0. Multi-factor authentication is available for administrator accounts.

## Access Control
Role-based access controls are available. Customer administrators can assign agent, manager, and admin roles.

## Encryption
Data is encrypted in transit using TLS. Data at rest is encrypted using cloud provider managed encryption.

## Logging and Monitoring
BrightDesk maintains audit logs for administrative activity. Standard plans retain logs for 90 days.

## Incident Response
BrightDesk maintains an incident response process. Customers are notified of confirmed incidents affecting their data. The questionnaire does not provide a specific notification timeframe.

## Compliance
BrightDesk has completed an internal security review. A SOC 2 Type II report is not currently available.

## Data Retention
Deleted tickets are retained in backup systems for up to 90 days before permanent deletion.

## Business Continuity
BrightDesk performs daily backups and maintains disaster recovery procedures.
